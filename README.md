## Futures
- Save posts as bookmarks
- Mute certain topics
- Able to load previous posts offline
- Download images
- Copy or launch URLs

## Create
First clone the project
```powershell
git clone 'https://github.com/empitrix.com/mosfet_app'
cd ./mosfet_app
flutter create .
```
